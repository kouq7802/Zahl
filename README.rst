RepeatingDecimal.py
-------------------
This program can find period length of repeating decimal.

Every repeating decimal can be written as a fraction.

HOW TO USE
----------
Open in terminal this repository and type below.

.. code-block:: bash
        
        $ ./RepeatingDecimal.py <denominator> <numarator>
